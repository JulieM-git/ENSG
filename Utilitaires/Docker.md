# docker

### Utilisation du docker-compose ###
```
docker-compose up -d
```

### Trucs et astuces ###
- Supprimer tous les conteneurs
```
docker rm $(docker ps -a -q) -f
```
- Supprimer toutes les images
```
docker rmi $(docker images -q) -f
```
- Version du docker-compose
```
docker-compose --version
```
